## 파일 구조
```
quote-app/
├── index.html              # 메인 HTML (폼 + 미리보기)
├── css/
│   └── style.css           # 전체 스타일 (폼 + 견적서 레이아웃)
├── js/
│   ├── config.js           # 가격 설정, 옵션 목록
│   ├── form.js             # 폼 동적 처리, 유효성 검사
│   ├── calculator.js       # 가격 계산 로직
│   ├── preview.js          # 견적서 미리보기 렌더링
│   ├── pdfExport.js        # PDF 생성/다운로드
│   └── app.js              # 메인 진입점, 모듈 통합
└── assets/
    └── logo.png            # 회사 로고 (선택)
```
